# Bookmark Landing Page

A responsive landing page for a simple bookmark manager application, built with **HTML**, **CSS**, and **Bootstrap**.  

## Features

- Fully responsive layout for desktop and mobile.
- Hero section with illustration and call-to-action buttons.
- Features section with interactive buttons.
- Browser extension download cards.
- Frequently Asked Questions (FAQ) section using Bootstrap Accordion.
- Contact form to subscribe via email.
- Footer with navigation links and social media icons.
s

Email validation is handled with HTML5 native validation, no JavaScript needed.

Fully responsive and tested on multiple screen sizes.
